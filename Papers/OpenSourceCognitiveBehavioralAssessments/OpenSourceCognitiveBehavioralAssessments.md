# Open-Source Cognitive & Behavioral Assessment Tools

Research review compiled 2026-06-29. Tools are grouped into: **(A)** ready-to-administer batteries with built-in standardized tasks, **(B)** general-purpose experiment-builder frameworks (no tasks out of the box, but the dominant infrastructure most batteries are built on), and **(C)** specialized platforms (mobile/applet-based, adaptive testing).

## Notable mention (not strictly open-source code, flagged for completeness)

- **R-CARE Toolbox** (NIH-funded remote neuropsych battery for aging/Alzheimer's research) is built on **REDCap**, which is free for academic/non-profit use but not OSI-licensed open source — worth knowing about given a clinical-aging research focus, but doesn't meet a strict "open source" bar. Online administration: yes, via a REDCap instance, typically institutionally hosted.

## Observed patterns

- jsPsych is the de facto backbone for most modern browser-based batteries (TestMyBrain, Self-Regulation Ontology, Experiment Factory, and the Niv Lab demos all sit on top of it).
- None of these tools ship with serious built-in multi-language UI out of the box — "language of administration" almost always means "whatever you type into the config." PEBL and MindLogger have the most documented community-translation workflows.
- Licensing is inconsistent across academic repos: well-maintained infrastructure projects (jsPsych, PsychoJS, OpenSesame, PsychoPy, lab.js, Concerto, Expyriment) carry clear OSI licenses, but several research-lab batteries (Self-Regulation Ontology, ICAR) have no LICENSE file at all — the code is publicly visible on GitHub but not formally open-sourced.
- "Online administration" splits into three hosting models: (1) centrally hosted by the project itself (PEBL Hub, TestMyBrain, PsyToolkit, MindLogger's CMI infrastructure), (2) self-hosted by the researcher on their own server/Docker container (Experiment Factory, Flowers Team battery, lab.js, OpenSesame via JATOS, Concerto), or (3) desktop/local-only with no online path (Expyriment, Psychtoolbox-3, PsyBuilder, cognitive-battery).
